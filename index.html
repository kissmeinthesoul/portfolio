Introduction

We will work in the Figma. Figma is the application for web-designers that allows creating amazing mockups, ux & ui design from the website up to mobile and clocksuoi8. Firstly, all are free. Secondly, this program is comfortable for team working. Thirdly, you can use the app in the browser without installation. It’s time to Start!

Body. Registration

In the beginning, you must follow www.figma.com and register by clicking the button "Sign up" or "Try Figma for free". Type the email and password, then click the button "Create account". Also, you can register by clicking the button "Sign up with Google" if you have an email from service "gmail.com". Then choose "Design" in the dropdown list and click the button "Create account". On the popup window, you should click on "Do this later" because we don’t need to create a team for now. At all the prompts, click "next." Excellent, we created an account.

Body. Preparation

Then we should choose "Frame" that we will use. The frame is the content area, desktop, mobile, tablet, for instance. In this lesson, we will make a mockup for desktop, so you should choose the desktop, then change the width to 1920px and height to 1080px. Now we can start editing our mockup.

Body. Main

Before recording, I found pictures and invented text. You can follow www.unsplash.com to choose own pictures for the project. Ok, start. We should move the main picture to frame, then change the width of a picture to 1920px and beautifully correct the picture. As we see, my photo is very bright, so we choose the item "rectangle" and put the width and height as of our frame(1920:1080). Change color scheme from solid to the gradient. Next, you should choose points of color grading scale and change color and opacity, i.e. follow my actions on the screen… Cool work! Now, our sun not so bright as before and we can create a logo through the simple text. Choose the item "Text" then you may edit characteristics as you want. All fonts uploaded by fonts.google.com is available. Also, you can choose the size, weight, letter-spacing, align-spacing, position and etc... Ok. We created a logo. Thereby, we should make a menu, but choosing the size smaller… Now, we have to create a headline, the headline must be purpose of the site or company... As you know, we didn't add the button to join. Let's do this! Create a small rectangle with size as you want and put text on it… More than half of the work is done. We have to do blocks of the text "Address" and "Contacts"...
